# Introduce

Convert numbers to cardinal numbers

## Usage
``` javascript
import { getCardinalDescription } from 'cardinal-convert';

const str = getCardinalDescription(1000) // cardinal str
```

## Support language
- [x] Spanish
- [ ] Chinese
- [ ] English
